BohConf is the official RailsConf 2010 unconf. At BohConf, we're going to get our hands dirty writing code and sharing ideas in an open and free-form environment. This hacking-centric event will include community code drives featuring well known OSS authors, a code retreat, a programming competition, and barcamp-style discussions. It is free and will run alongside RailsConf in the convention center. Everyone is welcome at BohConf.
